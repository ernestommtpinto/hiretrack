# HireTrack — Premium Job Application Tracker

HireTrack is a responsive React application created by **Ernesto Pinto** to help candidates organize their job search with a clean, professional dashboard.

The project was designed as a portfolio-ready product rather than a basic demo. It shows practical React skills, component architecture, Bootstrap UI implementation, local data persistence and recruiter-friendly attention to detail.

## Features

- Add new job applications with company, role, status, salary, contact, link and notes
- Filter applications by status
- Search by company, role or location
- Sort by newest, oldest or company name
- Dashboard cards with live statistics
- Application table with clean recruiter-focused layout
- Status progress insights and monthly activity preview
- Data persistence with `localStorage`
- Fully responsive layout built with Bootstrap and custom CSS
- Polished sidebar, cards, forms and dashboard interface

## Getting Started

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
